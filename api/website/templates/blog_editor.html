La Salle Center Blog Editor

Create or delete blog posts.

Create Blog Posts

Blog posts are a way to relay information to parents, students, and even potential students. Make sure your blog post is concise yet informative.

Before creating your post, make sure you have proofread it and recieved approval from your center director to upload it.

Delete Blog Posts

If a mistake was made or a blog post didn't work out, then delete it. the blog post ID can be found under the post's content.

WARNING! Deleting a blog post is a permanent action! If you want to edit a blog post, then copy the text from the post you want to edit and create a new post. Then, delete the old post.
